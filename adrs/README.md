# ADRs (Micro-Decisões)

Esta pasta registra micro-decisões operacionais e de governança que evitam retrabalho e podem ser reaplicadas em outros projetos.

Use ADR quando a decisão:

- aumentar segurança ou governança
- alterar o fluxo humano + IA
- evitar erro recorrente
- gerar padrão reutilizável

Formato recomendado:

- contexto
- decisão
- ação executada
- resultado observado
- lição reutilizável
- referências
