# AGENTS.md

Este arquivo define o fluxo obrigatorio para qualquer IA neste repositorio.

## Regras obrigatorias de execucao

1. Antes de editar qualquer arquivo, confirmar branch atual.
2. Se estiver em `main`, criar branch dedicada e so depois iniciar mudancas.
3. Fazer commits incrementais por unidade logica (docs, timeline, iniciativas, automacao), sem agrupar mudancas nao relacionadas.
4. Nao versionar artefatos operacionais locais (ex.: `bp-*.json`, payloads temporarios).
5. Abrir PR sempre em modo Draft, com titulo e descricao completos.
6. Incluir issue vinculada quando aplicavel e explicitar objetivo editorial/operacional do PR.
7. Converter Draft para Ready somente apos validacao da Rosana.
