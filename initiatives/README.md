# Initiatives

Cada pasta em `initiatives/` resume uma frente importante do projeto.

Use estas páginas para entender:

- qual problema estava em jogo
- por que a iniciativa virou prioridade
- qual decisão foi tomada
- qual impacto era esperado

As iniciativas são o melhor ponto de entrada para leitura rápida em entrevistas, conversas com gestores e contexto executivo.
